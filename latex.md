## Отношения

| Символ    | LaTeX   | Обозначает             |
| --------- | ------- | ---------------------- |
| $\subset$ | \subset | Является подмножеством |

## Логические операции

| Символ    | LaTeX   | Обозначает                                 |
| --------- | ------- | ------------------------------------------ |
| $\forall$ | \forall | Для любого                                 |
| $\exists$ | \exists | Существует (найдется) по крайней мере один |

## Множества

| Символ | LaTeX | Обозначает         |
| ------ | ----- | ------------------ |
| $\in$  | \in   | Является элементом |
